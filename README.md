# orchard releases

Prebuilt binaries for the [Orchard](https://orchard.infra.hackclub.com) CLI.
Published automatically by goreleaser from the (private) source monorepo on every cli release.

Install:

```sh
brew tap leafdtk/tap
brew install --cask orchard-cli
```

or:

```sh
curl -fsSL https://raw.githubusercontent.com/LeafdTK/orchard-releases/main/install.sh | sh
```
